# Git Backup Script для AnonChat

Автоматический скрипт для создания Git бэкапов проекта AnonChat.

## 🚀 Использование

### PowerShell (рекомендуется для Windows)

```powershell
# С автоматическим сообщением коммита
.\git-backup.ps1

# С пользовательским сообщением
.\git-backup.ps1 "🔥 Обновлён ChatCore"

# С эмодзи и описанием
.\git-backup.ps1 "🛡️ Улучшена система защиты"
```

## ✨ Возможности

- ✅ **Автоматическое добавление** всех изменений
- ✅ **Автогенерация сообщений** коммитов с временными метками
- ✅ **Цветной вывод** статусов в консоль
- ✅ **Подробное логирование** в `backup.log`
- ✅ **Проверка Git статуса** и ветки
- ✅ **Обработка ошибок** с понятными сообщениями
- ✅ **Автоматический пуш** в текущую ветку

## 📋 Примеры сообщений коммитов

```powershell
.\git-backup.ps1 "🎨 Обновлён дизайн чата"
.\git-backup.ps1 "🐛 Исправлена ошибка отправки сообщений"
.\git-backup.ps1 "⚡ Оптимизирована производительность"
.\git-backup.ps1 "🆕 Добавлена новая тема"
.\git-backup.ps1 "🔧 Рефакторинг ChatCore"
```

## 📊 Логирование

Все операции записываются в файл `backup.log`:

```
[2025-08-06 22:40:25] [INFO] === Starting Git backup for AnonChat ===
[2025-08-06 22:40:26] [INFO] Current branch: main
[2025-08-06 22:40:26] [INFO] Creating commit: 🛡️ Improved ChatCore protection system
[2025-08-06 22:40:26] [SUCCESS] Commit created successfully
[2025-08-06 22:40:31] [SUCCESS] Changes pushed successfully
[2025-08-06 22:40:31] [SUCCESS] === Git backup completed successfully ===
```

## 🛡️ Безопасность

- Проверка наличия Git репозитория
- Валидация текущей ветки
- Обработка исключений
- Проверка статуса операций Git
- Автоматический откат при ошибках

## 🎯 Статусы выполнения

- `[SUCCESS]` - Операция выполнена успешно
- `[ERROR]` - Критическая ошибка
- `[WARNING]` - Предупреждение (нет изменений)
- `[INFO]` - Информационное сообщение

## 🔧 Требования

- PowerShell 5.1+ или PowerShell Core
- Git установлен и настроен
- Доступ к GitHub репозиторию
- Права на запись в текущую директорию

## 🚨 Устранение проблем

### Ошибка "Git repository not found"
```powershell
# Убедитесь, что вы в корне проекта
cd C:\Users\vkuli\anonchat-mobile
```

### Ошибка "Could not push changes"
```powershell
# Проверьте настройки Git
git config --list
git remote -v
```

### Проблемы с правами
```powershell
# Запустите PowerShell от имени администратора
# Или измените политику выполнения
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

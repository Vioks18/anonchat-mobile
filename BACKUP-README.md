# 🔄 Система восстановления AnonChat

## 📁 Резервные копии

- **`app/screens/ChatScreen.backup.tsx`** - Резервная копия рабочей версии
- **`restore-backup.js`** - Скрипт для автоматического восстановления

## 🚨 Как восстановить рабочую версию

### Способ 1: Автоматическое восстановление
```bash
node restore-backup.js restore
```

### Способ 2: Ручное восстановление
1. Скопируйте содержимое `app/screens/ChatScreen.backup.tsx`
2. Вставьте в `app/screens/ChatScreen.tsx`
3. Сохраните файл

### Способ 3: Через терминал
```bash
cp app/screens/ChatScreen.backup.tsx app/screens/ChatScreen.tsx
```

## 🔧 Создание новой резервной копии

```bash
node restore-backup.js backup
```

## ⚡ Встроенная защита

В `ChatScreen.tsx` добавлена обработка ошибок:
- При любой ошибке автоматически загружается резервная версия
- Консоль покажет сообщение "Резервная версия загружена"

## 🚀 **Как запустить:**

### 1. Настройка Firebase
```bash
# Следуйте инструкциям в FIREBASE-SETUP.md
# Обновите конфигурацию в app/services/firebase.ts
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Запуск
```bash
npm start
```

### 4. Тестирование на двух устройствах
- Устройство 1: сканируйте QR код
- Устройство 2: сканируйте QR код
- Создайте чат между устройствами по UID

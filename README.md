# GameGuardian Scripts

Коллекция Lua скриптов для GameGuardian

## 📁 Файлы

### `amongus_autohack.lua`
- **Назначение:** Автопоиск для Among Us
- **Функции:**
  - Поиск скорости игрока
  - Модификация времени задач
  - Изменение прогресса
  - Поиск изменяющихся значений

### `universal_autohack.lua`
- **Назначение:** Универсальный автопоиск для любых игр
- **Функции:**
  - Универсальный поиск значений
  - Экспериментальный взлом
  - Поиск по изменению
  - Случайная модификация

## 🚀 Как использовать

1. **Установите GameGuardian:**
   - Скачайте с официального сайта
   - Установите на рутованный телефон или эмулятор

2. **Запустите игру:**
   - Откройте игру которую хотите модифицировать
   - Убедитесь что игра работает

3. **Запустите GameGuardian:**
   - Откройте GameGuardian
   - Выберите процесс игры
   - Нажмите на значок скрипта

4. **Загрузите скрипт:**
   - Скопируйте код из .lua файла
   - Вставьте в GameGuardian
   - Запустите скрипт

## ⚠️ Предупреждения

- **Используйте только для обучения!**
- **Игра может сломаться** - перезапустите если что-то пойдет не так
- **Некоторые игры имеют защиту** - скрипты могут не работать
- **Многопользовательские игры** - риск бана!

## 🎯 Рекомендации

- Начинайте с офлайн игр
- Тестируйте на копиях игр
- Изучайте как работают скрипты
- Не используйте в популярных онлайн играх
